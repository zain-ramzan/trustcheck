# TrustCheck
Lightweight assurance layer to score the reliability of LLM-generated intents using round-trip checks and heuristics.

## Quick start
1. Install reqs: `pip install -r requirements.txt`
2. Ensure Intentify's `translate.py` is accessible or adjust imports.
3. Run: `streamlit run app.py`

## Files
- app.py: Streamlit frontend
- check.py: verification utilities
- requirements.txt
